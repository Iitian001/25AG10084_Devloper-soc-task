# Campus Connect Platform 🎓

Hello! I'm **Shreyash Mishra**, a Full-Stack Developer and AI Automation Expert.

This repository contains my task for the Developers' Society, where I engineered a complete, full-stack **Campus Connect Platform**. The platform is designed to be a central hub for campus notices, events, and updates, featuring a bold, highly-responsive Neo-Brutalism design system.

## 💻 Tech Stack
The platform is built using a modern, scalable tech stack:

- **Frontend:** [Next.js](https://nextjs.org/) (React Framework)
- **Backend:** [Node.js](https://nodejs.org/) & Express
- **Database:** [Supabase](https://supabase.com/) (PostgreSQL + Auth)
- **Deployment:** 
  - Frontend hosted on **Vercel**
  - Backend API hosted on **Render**
- **Styling:** Tailwind CSS with a custom Neo-Brutalism design architecture

## 🌟 Key Features
- **Admin Dashboard:** Secure, role-based authentication allowing administrators to post new notices and events directly from the web interface.
- **Real-Time Feed:** A dynamic feed of campus events and notices fetched directly from the Node.js backend.
- **Dynamic Routing:** Beautifully styled detail pages for every notice and event.
- **Responsive Design:** Completely mobile-optimized layout that seamlessly scales across desktop and mobile devices without horizontal overflow.
